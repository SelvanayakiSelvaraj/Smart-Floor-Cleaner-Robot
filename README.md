 📖 Project Description

The Smart Floor Cleaner Robot is an automated cleaning system designed to remove dust and small particles from the floor. It uses a rotating fan mechanism to collect dust and an obstacle detection sensor to avoid collisions while moving.

The robot can move automatically in different directions and detect objects in its path. This project helps in understanding automation, embedded systems, sensors, and robotic movement.

⚙️ Working Principle

The robot moves automatically on the floor.

The rotating fan creates suction or pushes dust into the collector area.

Sensors continuously detect nearby obstacles.

When an obstacle is detected, the robot changes direction automatically.

The cleaning process continues until stopped manually.
