Smart Floor Cleaner Robot
📖 Project Description

The Smart Floor Cleaner Robot is an automated cleaning robot designed to collect dust and small particles from the floor using a rotating fan mechanism. The robot moves automatically and uses an ultrasonic sensor to detect and avoid obstacles.

The rotating fan helps pull or push dust into the collection area while the robot navigates around objects without human control.

🚀 Features

Automatic floor movement

Dust collection using rotating fan

Obstacle detection and avoidance

Autonomous navigation

Low-cost smart cleaning system

⚙️ Working Principle

The robot moves automatically using DC motors.

A rotating fan collects dust from the floor.

The ultrasonic sensor continuously checks for obstacles.

If an obstacle is detected, the robot changes direction automatically.

The cleaning process continues until the power is turned off.
